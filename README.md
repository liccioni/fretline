# Fretline – Guitar Practice App

Fretline is a focused guitar practice tool designed for real daily use. Build and run **custom practice sessions** with multiple drills, including note location, triad exercises, spider/finger drills, and generic metronome-based exercises.

---

## Features

- Build practice sessions of any length (20–60 minutes)
- Mix multiple drill types in a single session
- Automatic drill transitions with countdowns
- Audible metronome clicks with accent on the downbeat
- Fullscreen, distraction-free session runner
- All state in memory, no backend required
